class thesoup::types::IsTemplateSpecialization< class, class >

Static asserter to see if a type is specialization of a template.

Template Parameters
TThe derived type.
UThe original type.

Sometimes there might be a situation where we need to static assert that a type derives from a certain template. In those situations, we will be able to use this struct. See example below. For a real like use case, observe thesoup::algorithms::kruskal .

template <typename T> struct Base;
using ChildInt = Base<int>;
thesoup::types::IsTemplateSpecialization<ChildInt, Base>::value; // will be true
thesoup::types::IsTemplateSpecialization<char, Base>::value; // will be false
