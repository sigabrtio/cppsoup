Detailed Description

template<typename T>
class thesoup::types::Slice< T >

A class akin to python/rust Slice type (a fat pointer).

Template Parameters
TThe pointer type.

The Slice<T> type is meant to provide functionality similar to Rust's fat pointer types. It has 2 fields, a start pointer of type T, and a size. It provides some convenience overloads:

  • Overloaded [] operator for indexing with range checking.
  • Overloaded begin and end methods to enable range for loop.

NOTE: Move and move assignment has been disabled to avoid multiple ownership mess.
