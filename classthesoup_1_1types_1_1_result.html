Detailed Description

template<typename T, typename E>
class thesoup::types::Result< T, E >

A class akin to Rust's Result<T, E>. Helps us avoid having to throw exceptions. The Result<T, E> type is used to return the results of operations in the library.

Template Parameters
TThe success type.
EThe error type.

Usage:

Result res {Result<int, int>::success(1)}; Result res2{Result<int,int>::failure(2)}; std::cout << res.unwrap() << "\n"; std::cout << res2.error() << "\n"; if (res) { std::cout << "Yes\n"; } else { std::cout << "No\n"; }
